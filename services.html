Services Provided by the Application

1. Search for Hospitals

The Hospital Finder application allows users to search for hospitals in a specific location. Users can enter a city or zip code to find hospitals nearby.

2. Filter Hospitals

The application provides filters to narrow down search results based on the type of hospital, services provided, and ratings. Users can also sort the results by distance, ratings, or popularity.

3. View Hospital Details

Users can view details about each hospital, including the address, phone number, website, and reviews from other users.

4. Get Directions

The application provides directions to the hospital from the user's current location using Google Maps.

5. Save Favorites

Users can save their favorite hospitals for easy access in the future. They can also leave reviews and ratings for the hospitals they have visited.
